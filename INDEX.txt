================================================================================
                         CONFIGURATION
================================================================================

Default locations (configurable via Settings menu):

CONFIG_DIR=/docker
- Where all container configs are stored
- Each app gets its own subdirectory

MEDIA_DIR=/mnt/media
- Shared media library
- Default structure: /mnt/media/{movies,tv,downloads,music,books}

TZ_VAL=America/New_York
- Timezone for all containers
- Change via Settings menu

PUID_VAL=0, PGID_VAL=0
- User/Group IDs (0=root)
- Change via Settings menu

================================================================================
                        DOCKER INTEGRATION
================================================================================

Auto-generated files:
- /docker/docker-compose.yml

Container management:
- All containers in single docker-compose project
- Uses depends_on for proper startup ordering
- Volume mapping for /config and /media
- Environment variables for PUID/PGID/TZ
- Restart policy: unless-stopped
- Proper network and privilege handling

Manual docker commands:
$ docker compose -f /docker/docker-compose.yml ps
$ docker compose -f /docker/docker-compose.yml logs -f appname
$ docker compose -f /docker/docker-compose.yml stop appname
$ docker compose -f /docker/docker-compose.yml up -d appname
